الجزء الأول — إصلاح كل أخطاء TypeScript:
1. شغّل: npx tsc --noEmit
2. أصلح كل خطأ يظهر (متوقّع ~59 خطأ):
   - أغلبها imports غير مستخدمة (TS6133): احذف الـ import غير
     المستخدم
   - الأخطاء المهمة المعروفة:
     * App.tsx (~السطر 47): نوع FallbackComponent في ErrorBoundary
       غير متوافق — اضبط النوع ليطابق توقيع react-error-boundary
     * RouteErrorFallback.tsx (~السطر 15): قيمة unknown تُمرّر
       كـ ReactNode — حوّلها لنص (String(error)) أو تحقّق نوعها
3. أصلح الأخطاء بشكل صحيح:
   - ممنوع منعاً باتاً: any / @ts-ignore / @ts-expect-error
   - للقيم unknown: استخدم type guard أو تحويل صريح
4. كرّر tsc --noEmit حتى 0 أخطاء
